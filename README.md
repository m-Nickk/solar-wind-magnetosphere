# Solar Wind & Magnetosphere Simulation

A 2D/3D particle simulation built with Pygame and Python. It models the behavior of charged solar wind particles interacting with Earth's magnetic dipole field via the Lorentz force.

## Features
- Real-time physics simulation using Euler integration.
- Custom magnetic dipole field calculation.
- Particle depth-sorting and motion trails for visual clarity.
- Dynamic particle respawning to simulate continuous solar flow.
